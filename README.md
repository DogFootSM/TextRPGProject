# TextRPG 프로젝트
 
# 1. 프로젝트 개요
- 프로젝트 이름 : TextRPG 제작 프로젝트
- 의도 : 디지몬 RPG를 콘솔 TextRPG 형식으로 구현
- 개발 인원 : 1인
- 개발 도구 : VisualStudio 2022
- 개발 언어 : C#
- 기능 구현 :
    - 캐릭터
    - 인벤토리
    - 상점
    - 전투
    - 아이템
- 총 개발 기간 : 24.08.12 ~ 24.08.16
 
# 2. 게임 규칙 
- 디지몬 RPG 게임을 콘솔 TextRPG 게임으로 구현
- 몬스터를 사냥하고 캐릭터를 성장 시키는 게임
